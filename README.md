## View slides locally

First, ensure you have the following installed:

* [Node.js][]
* [Gulp CLI][]

    npm install -g gulp-cli

Next, install dependencies (if not already installed by the generator):

    npm install

Finally, build and serve the presentation!

    gulp serve
